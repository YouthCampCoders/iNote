### 项目依赖工具

> 除了 react react-dom 等核心以外用到的包

- **yarn** 依赖管理
- **react-router-dom** 路由跳转
- **prettier** 代码风格统一
- **AntDesign** 样式库
- **less** CSS 拓展
- **craco** 项目配置
- **axios** 请求封装
- **redux** 状态管理

### 项目启动

```
git clone https://github.com/YouthCampCoders/iNote_FE.git
```

安装 yarn

```shell
npm install yarn -g
```

安装依赖

```shell
yarn install
```

启动项目

```shell
cd ./INote_FE
yarn start
```
